# Making API Calls

Due to sockets being unavailable in Pyodide, http libraries are [currently unsupported](https://pyodide.org/en/stable/project/roadmap.html#write-http-client-in-terms-of-web-apis) out of the box.

A patch for `requests` and `urllib` has been provided by the package [Pyodide-HTTP](https://github.com/koenvo/pyodide-http) and is preinstalled in `react-py`.

See [Using Packages](using-packages) for package installation instructions, for these examples the following `Packages` object has be used:

```js
{
  micropip: ['requests']
}
```

Try it out:

export const exampleCode = `import requests
response = requests.get('https://api.github.com/repos/elilambnz/react-py')
data = response.json()\n
print("*" * 50)
print(f"Repository: {data['full_name']}")
print(f"Owner: {data['owner']['login']}")
print(f"Number of stars: {data['stargazers_count']} ‚≠êÔ∏è")
print(f"Number of forks: {data['forks_count']} üç¥")
print(f"Number of watchers: {data['watchers_count']} üëÄ")
print("*" * 50)
`

## `GET`

export const exampleGetCode = `import requests
response = requests.get('https://jsonplaceholder.typicode.com/users/1')
data = response.json()\n
print(data)
`

## `POST`

export const examplePostCode = `import requests
response = requests.post('https://jsonplaceholder.typicode.com/posts', { "title": "My Post", "body": "This is the content of my post", "userId": 1 })
data = response.json()\n
print(data)
`

<CodeEditor code={examplePostCode} packages={{ micropip: ['requests'] }} />

## `PUT`

export const examplePutCode = `import requests
response = requests.put('https://jsonplaceholder.typicode.com/posts/1', { "id": 1, "title": "Updated Post", "body": "This is the updated content", "userId": 1 })
data = response.json()\n
print(data)
`

<CodeEditor code={examplePutCode} packages={{ micropip: ['requests'] }} />
